# EexCompilerViewer

App that allows you to step through the compilation of an EEx template using the various EEx Engines.

- EEx.SmartEngine
- Phoenix.HTML.Engine
- Phoenix.LiveView.Engine
- Phoenix.LiveView.HTMLEngine
